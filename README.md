# 📊 Excel Sales Data Analysis (Bootcamp Project)

This project was completed during my **Data Technician Bootcamp** and focuses on exploring and analysing sales datasets using **Microsoft Excel**.

The workbook demonstrates practical Excel skills used in data roles, including **data filtering, formula-based analysis, PivotTables, logical functions, and visualisation** to explore patterns in retail and sales data.

---

## 📁 Project Overview

In this project I worked with multiple sales datasets to practice real-world data analysis tasks such as:

- Cleaning and exploring datasets
- Performing calculations using Excel formulas
- Summarising data with PivotTables
- Categorising data with logical functions
- Creating charts to visualise insights

The exercises simulate common tasks a **Data Technician or Junior Data Analyst** might perform when analysing business data.

---

## 🛠 Excel Skills Demonstrated

### Spreadsheet Functions

The analysis uses a variety of Excel formulas to transform and analyse data:

- `SUM` – Calculating total commission and sales values  
- `SUMIF` – Conditional totals based on criteria  
- `AVERAGE` – Calculating averages across datasets  
- `AVERAGEIF` – Analysing averages for specific conditions  
- `DATE`, `MONTH`, `YEAR` – Working with time-based data  
- `UNIQUE` – Identifying distinct values in the dataset  
- `VLOOKUP` – Retrieving related information across tables  
- `SWITCH` – Categorising data based on conditions  

Example categorisation formula used in the project:

```excel
=SWITCH(TRUE, C2 > 600, "High", C2 >= 300, "Medium", "Low")
```

This formula categorises products based on **sales volume levels**.

## 📊 Data Analysis

### PivotTables

PivotTables were used to summarise and analyse datasets such as **bike sales and product sales by county**.

Examples of analysis performed include:

- Sales performance by **country**
- Market segmentation by **age group and gender**
- Product sales totals by **county**
- Comparison of **product categories**

### Key Finding

- **Female customers generated significantly higher profits than male customers overall.**
